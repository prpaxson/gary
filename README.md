## Installation
```
git clone https://github.com/prpaxson/gary.git
cd gary
make gary
```
If the above does not work (like on a lab computer),
try making an alias in your .bashrc or something:

`alias gary=/home/linux/ieng6/oce/59/(USERNAME)/gary/gary`
## Usage
```
gary
gary says
```
